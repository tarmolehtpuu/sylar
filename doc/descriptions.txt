------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.alphabet: Contains classes that define the alphabets for formulas and sequents.
------------------------------------------------------------------------------------------------------------------------

- Alphabet: super class for all alphabets, is used for parsing, generating and serializing formulas and sequents.
- AlphabetFormula: class defining a valid alphabet for formulas.
- AlphabetSequent: class defining a valid alphabet for sequents.

------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.formula: Contains the implementations for formulas and formula collections (sets of formulas).
------------------------------------------------------------------------------------------------------------------------

- Formula: class defining a formula, contains methods for evaluating the truth value of a formula.
- FormulaCollection: a class representing a set of Formulas, contains method for tautology and contradiction checks.
- FormulaException: generic exception thrown by classes in the formula package.
- FormulaFilter: interface for filtering formulas inside a FormulaCollection.

------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.generator: Contains classes for formula and sequent generation.
------------------------------------------------------------------------------------------------------------------------

- Generator: interface for all formula generators
- GeneratorAxioms: generator for sets of formulas that may not be contradictory.
- GeneratorException: generic exception thrown by classes in the generator package.
- GeneratorFormula: contains an implementation of random formula generator based on Markov chains.
- GeneratorSequent: contains an implementation of random sequent generator based on Markov chains.

------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.inference: Contains the interface and implementation for different inference strategies.
------------------------------------------------------------------------------------------------------------------------

- InferenceEngine: implementation of an InferenceEngine, which given a set of axioms and a set of known values, tries
                   to infer as much as possible (until no new knowledge can be inferred).
- InferenceException: generic exception thrown by classes in the inference package.
- InferenceHelper: helper class for inference strategies, contains helper methods for determining if given a set of
                   known values, all the rest of the interpretations of a formula evaluate constantly to true or false.
- InferenceRequest: implementation of a request to perform inference, contains an axiom and a set of known values.
- InferenceResult: contains the results of an inference strategy application, in case inference was successful it will
                   contain all the values for new knowledge that we managed to infer.
- InferenceStrategy: generic interface for inference strategies.
- InferenceStrategyEQ: contains an inference strategy implementation for equivalence.
- InferenceStrategyMP: contains an inference strategy implementation for Modus Ponens.
- InferenceStrategyMT: contains an inference strategy implementation for Modus Tollens.

------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.input: contains classes related to parsing formulas and sequents.
------------------------------------------------------------------------------------------------------------------------

- Lexer: Converts a string into tokens, which are later used by the Parser.
- LexerException: Generic exception thrown by Lexer methods.
- Parser: Super class for all parsers, which convert strings into either formulas or sequents.
- ParserException: Generic exception thrown by Parser methods.
- ParserFormula: Parser implementation for formulas.
- ParserSequent: Parser implementation for sequents.
- Token: Represents a valid token returned by the Lexer, later used by the parser.
- TokenType: Enum that lists all the valid token types.


------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.output: contains classes related to converting a formula into another format (for example SVG or .dot).
------------------------------------------------------------------------------------------------------------------------

- Serializer: Provides an interface for all formula instances.
- SerializerDot: Serializes a formula into a .dot file (for GraphViz visualization).

------------------------------------------------------------------------------------------------------------------------
ee.moo.sylar.util: contains utility classes used by other packages.
------------------------------------------------------------------------------------------------------------------------

- Console: Class that provides the methods for use in Sylar's interactive console interface.
- PermutationIterator: Helper class for iterating over all possible interpretations of a formula (truth table).
- Stack: Contains a fast stack implementation.
- StringUtil: Helper methods for dealing with strings.
